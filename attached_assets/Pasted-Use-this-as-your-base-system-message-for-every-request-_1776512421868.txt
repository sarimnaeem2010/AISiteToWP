🧠 5. COST CONTROL STRATEGY (VERY IMPORTANT)

To minimize credits:

Chunk requests (don’t send full site)
Cache results in DB
Run AI only:
on upload
on user request (not real-time)
❌ AVOID:
sending full HTML
sending CSS
continuous AI calls on every UI change
